    public getTilingTileData(area: Object, originX: number, originY: number, x: number, y: number): Object source 

    import {getTilingTileData} from 'tile-tools/src/index.js'

    Gets tiling data from a tile area.

    Most of the time when mapping tilemaps, having the pen be "tiling" is more useful than having it overwrite everywhere. This function provides that kind of functionallity.

    Params:

    NameTypeAttributeDescription
    area Object

    The tile area to execute the operation on.
    originX number

    The x coordinate to tile from.
    originY number

    The y coordinate to tile from.
    x number

    The x coordinate to get data from.
    y number

    The y coordinate to get data from.

    Return:

    Object

    The calculated tile.

    public mergeTileAreas(area1: Object, area2: Object, x: number, y: number, skipEmpty: boolean): * source 

    import {mergeTileAreas} from 'tile-tools/src/index.js'

    Merges two tile areas together, putting the second one on top of the first one.

    The merge does not fail if it goes outside of the tile area.

    Params:

    NameTypeAttributeDescription
    area1 Object

    The tile area to use as the base.
    area2 Object

    The tile area to put on top of the first tile area.
    x number

    The x coordinate of where to put the second area relative to the first area.
    y number

    The y coordinate of where to put the second area relative to the first area.
    skipEmpty boolean

    If the insertion should skip empty (-1) tiles.

    Return:

    *
